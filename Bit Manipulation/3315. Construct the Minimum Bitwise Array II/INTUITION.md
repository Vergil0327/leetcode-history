# Intuition

num = 2^0 + 2^1 + ... + 2^k = x | (x+1)
=> 首先`x`, `x+1`必定各為1**奇數**1**偶數**

再來:
```
如果num = 11111
那麼  x = 01111

如果num = 10111
那麼  x = 10011 => x | (x+1) == num
```

由於我們操作只能+1, 最小x為: 將末端連續111...111右移一位
這樣+1後兩者互相OR, 就會補上

ex.

```
target: 1010101011111

=>
   1010101001111
OR 1010101010000
=  1010101011111 = target
```
