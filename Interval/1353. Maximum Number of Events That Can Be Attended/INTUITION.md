首先:
1. 透過startDay = events[0]可知道我們可不可以參加這event
2. 直覺上來講, 在所有可參加的event裡, event期間越短, 越早結束的優先參加 => min heap

所以我們可以遍歷每一天, 如果可以參加了, 就全加入到minHeap裡然後挑一個最早結束且還沒結束的的來參加, 並從min heap中pop掉
4, 最後計算總共參加多少events (亦即總共pop掉多少events)即可
